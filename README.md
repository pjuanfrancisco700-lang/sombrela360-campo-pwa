# Sombrela 360 — Campo (PWA)

1) Sube el contenido de `pwa_campo/` a un repo en GitHub.
2) En Settings → Pages: Source = Deploy from a branch → main → /(root).
3) Abre la URL de GitHub Pages en tu teléfono.
4) iPhone (Safari): Compartir → **Añadir a pantalla de inicio**.
   - El icono sale de `apple-touch-icon.png` (raíz).
5) Android (Chrome): Menú → **Instalar aplicación**.

Si no ves el logo en iOS, borra el acceso anterior, cierra Safari y vuelve a añadirlo (Safari cachea el icono).
